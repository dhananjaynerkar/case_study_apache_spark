# Data Folder

Place the provided case-study CSV datasets in this folder before running the notebook or Docker container.

Dataset source link:

```text
https://figshare.com/articles/dataset/OULAD_Open_University_Learning_Analytics_Dataset/5081998?file=8606371
```

Required local files:

- `assessments.csv`
- `courses.csv`
- `studentAssessment.csv`
- `studentInfo.csv`
- `studentRegistration.csv`
- `studentVle.csv`
- `vle.csv`

These CSV files are intentionally ignored by Git because they are input datasets and `studentVle.csv` is very large.
